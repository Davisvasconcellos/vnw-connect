# 🌐 VNW Connect - Plataforma de Conexão Social

Uma plataforma digital desenvolvida para conectar iniciativas sociais, voluntários e pessoas interessadas em gerar impacto positivo na sociedade. O projeto foca em uma experiência moderna, acessível, responsiva e inspiradora.

## 🌍 Contexto do Projeto

A plataforma tem como objetivos principais:
- Dar visibilidade a projetos sociais.
- Apresentar informações claras e acessíveis.
- Inspirar engajamento e transformação social.
- Facilitar a conexão entre voluntários, organizações e apoiadores.
- Transmitir confiança, empatia e profissionalismo por meio do design.

## 🚀 Tecnologias Utilizadas

- **React.js**: Biblioteca principal para construção da interface.
- **React Router DOM**: Gerenciamento de rotas e navegação.
- **SCSS (Sass)**: Estilização modular, utilizando o sistema de módulos moderno (`@use`) para compatibilidade e performance.
- **Vite**: Ferramenta de build rápida e moderna.

## 🧩 Estrutura e Funcionalidades

O projeto foi organizado com uma arquitetura modular e responsiva, contendo as seguintes páginas:

- **/** (Página Inicial): Apresentação da plataforma e impacto social.
- **/doacao**: Página dedicada a doações para causas diversas.
- **/voluntariado**: Oportunidades para voluntariado em projetos (Reciclagem, Tecnologia, Esporte).
- **/mentoria**: Conexão para mentorias profissionais, acadêmicas e de vida.
- **/eventosEP**: Divulgação de eventos, palestras e workshops.
- **/usuario**: Perfil do usuário voluntário.

## 🎨 Design System

O projeto segue uma identidade visual que transmite:
- 💙 **Confiança e Tecnologia**: Tons de azul (`#205691`, `#1a4575`).
- 🧡 **Impacto Social e Esperança**: Detalhes em laranja (`#FF5722`).
- 🤍 **Simplicidade e Acessibilidade**: Fundo branco e tipografia limpa.

**Tipografia**:
- *Inter* (Textos gerais)
- *Source Sans 3* (Títulos e botões)

## 📱 Responsividade

A aplicação é totalmente responsiva, adaptando-se a diferentes tamanhos de tela (mobile, tablet e desktop) utilizando:
- Unidades relativas (`rem`, `%`, `vh`, `vw`).
- Flexbox para layout fluído.
- Media Queries para ajustes específicos.

## 🛠️ Como Rodar o Projeto

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/Davisvasconcellos/vnw-connect.git
    cd vnw-connect
    ```

2.  **Instale as dependências**:
    ```bash
    npm install
    ```

3.  **Execute o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```

4.  Acesse `http://localhost:5173` no seu navegador.
